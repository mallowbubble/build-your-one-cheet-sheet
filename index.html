HTML Form Reference

Form Tags

Tag Description
<Form> Used to create an HTML form for user input
<Label> Defines a label for several elements
<Input> Specifies an input field where the user can enter data
<Textarea> Defines a multi-line text input control. is often used in a form, to collect user inputs like comments or reviews
<Button> Defines a clickable button
<Select> Used to create a drop-down list
<Option> Defines an option in a select list. Go inside a <Select>, <Optgroup>, or <Datalist> element
<Optgroup> Is used to group related options in a <Select> element (drop-down list)
<Fieldset> Is used to group related elements in a form. Draws a box around the related elements
The <Legend> tag is used to define a caption for the <fieldset> element.
<Output> Used to represent the result of a calculation (like one performed by a script)
